    介绍

    该仓库主要是为了研发文档的编写服务。 目前包含了三部分内容：

    研发文档编写指南

    目标是输出《研发文档编写指南》， 见：链接

    docfx的自定义模板

    为了扩展docfx v2的展示能力， 目前具有如下扩展：

    • 初步支持中文的全文搜索。（但还是有点弱）

    • 支持微软风格的Alerts标签， 见 ：链接

    • 支持mermaid的展示及pdf导出。

    docfx插件开发

    目前只包含了一个支持mermaid展示及导出的插件（该插件需要结合docfx的自定义模板实现）。

    使用

    使用《研发文档编写指南》

    前置条件： 安装 choco 工具， 参考这里

    Step 1: 安装docfx

    choco install docfx

    Step 2: 编译站点 切换当前目录到仓库根目录

    docfx build

    Step 3: 运行web服务 切换当前目录到仓库根目录。 该命令仅仅将_site目录作为web站点发布。

    docfx serve _site

    Step 4: 打开浏览器， 输入：http://localhost:8080/
