    表格

    以下为使用表格的基本要求：

    1. 如果可以用其它标签，则尽量不使用表格。
    2. 同类型表格必须使用同样的表格样式。 > 参数/信号/报警等类型的信息，必须使用表格。
    3. 不要出现空白表格，无需内容的，统一填写 “-” 。
    4. 标题要用名词，不要用句子。
    5. 出现多个单元格内容一样的，将内容复制，不要写 “同上” 。
    6. 同一列信息，表述方式一致，结尾标点符号也要一致。
    Note

    为确保表格的一致性，要求通过 Learn Markdown 插件的功能进行格式化。即通过插件里面的选项 Consolidate selected table 去除创建表格时额外多余的空格。

    具体使用参考章节：仓库及发布工具->VS Code->支持格式化表格。
