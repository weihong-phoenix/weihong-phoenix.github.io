    提示&警告

    是指出现在数控状态栏、用于告知操作员操作结果、当前进行操作、当前正在发生的报警或错误等操作提示类信息。按提示内容和提示目的，主要分为以下几类：

    • 提示正在进行或者即将开始的操作
    • 提示操作结果
    • 提示需要重新操作：操作结果非理想情况，需要用户重新操作。
    • 提示检查
    • 提示错误

    提示正在进行或者即将开始的操作

    尽量套用句型： [主语+] [正在] 动作描述......

    正例 反例
    N轴正在回机械原点......
    开始对刀......
    等待对刀信号......    		 N/A

    提示操作结果

    • 统一使用冒号代替中文“是、为”，提示操作结果。以数值结尾，句末无需标点符号。

    • 尽量套用句型：主语 + 操作结果描述: 结果值。

      正例 反例
      本次对刀完成，对刀结果：XXXX
      N轴粗精定位信号间距离： %0.3f      		 本次对刀完成，对刀结果是XXXX
      N轴粗精定位信号间距离为 %0.3f

    提示需要重新操作

    套用句型：结果/原因 + 请重新 + 操作描述 + ！

    说明

    • 统一使用感叹号结尾
    • 在前半句交代需要重新操作的原因或上次操作失败的结果。
    正例 反例
    N轴精定位误差过大，请重新回机械原点！ N/A

    提示检查

    套用句型：出错原因描述，请检查！

    正例 反例
    刀具号不合法，请检查！ N/A

    提示错误

    尽量套用句型：错误类型！+ 错误原因或解决方法。

    正例 反例
    参数设置错误！请确保工件直径是刀具直径的2倍或以上。
    参数设置错误！N轴螺距补偿参数"测量点数N"的设置值超出合理范围。    		 N/A

    〉例外：当前为非空闲状态，无法执行该操作！

    提示确认（待定）

    • 以对话框的形式提示，尽量套用句型：确认 + 操作 + 内容。
    • 正向结果使用确定按钮，反响选择使用取消按钮，按钮文字不使用是和否。
    正例 反例
    确认删除选中数据？ 你是否确认删除选择数据吗？

    提示描述

    避免用人称

    对于出错信息提示不要在句中采用第二人称，以免让读者感到受到责备。在适当的地方可采用被动语态。

    正例 反例
    登录 NcStudio Generator 失败。 您登录 NcStudio Generator 失败！

    和软件一致

    文档中返回信息同产品GUI返回的消息框中的信息提示保持一致。

    标点符号

    • 在界面设计中，消息统一处理为句子，句尾主要使用句号、问号、感叹号。

    • 在句子中，使用双引号。

    • 表格单元中只有GUI返回信息时，不加双引号。

    其它

    • 消息含义清晰，需提供被指对象的名称、场所。

    • 避免在消息中录入用户看不见的细节内容，比如以“*”显示或不显示的密码等。

    其它案例参考

    操作成功

    中文 英文 例句 备注
    做XXXX操作成功 Doing XXXX succeeded. 中文：保存数据成功。
    英文：Saving data succeeded.
    Succeeded in doing XXXX. 中文：删除生成日期小于或等于“1%”的历史任务成功。
    英文：Succeeded in deleting the history tasks with the creating date earlier than or equal to “1%”.    		 操作内容太长时使用此句型。
    XXXX succeeded. 中文：重新登录成功！
    英文：Re-login succeeded.    		 操作内容是一个名词或名词短语时使用此句型。

    操作失败

    中文 英文 例句 备注
    做XXXX操作失败 Doing XXXX failed. 中文：获取报表失败。
    英文：Obtaining reports failed.
    Failed to do XXXX. 中文：删除生成日期小于“1%”的历史任务失败。
    英文：Failed to delete the history tasks with the creating date earlier than “1%”.
    XXXX failed. 中文：重新登录失败。
    英文：Re-login failed.

    继续提示

    中文 英文 例句 备注
    确定要进行XXXX操作吗？ Are you sure you want to do XXXX? 中文：确定要删除当前记录吗？
    英文：Are you sure you want to delete the current records?    		 若受限于显示字符长度，该句式可调整为”Are you sure to do XXXX?”
    是否（想）要做XXXX操作？ Do you want to do XXXX? 中文：是否保留对此文档作的修改？
    英文：Do you want to save the changes to the file?

    操作超时

    中文 英文 例句 备注
    请稍后再试！ Please try later. 中文：服务器忙，请稍后再试！
    英文：The server is busy now. Please try later.
    XXXX超过…… XXXX exceeded…
    或XXXX have exceeded…    		 中文：登录失败超过三次。
    英文：The number of login attempts exceeds…
    Login failure has exceeded 3 times.（不推荐）    		 在同一产品中，其提示语的时态应该统一。
    XXXX操作超时 XXXX timed out.
    XXXX time out.    		 中文：操作超时。
    英文：Operation timed out.
    中文：等待超时。
    英文：Wait timeout.

    操作受限

    中文 英文 例句 备注
    做XXXX操作发生错误 XXXX error. 中文：NK300配置错误。
    英文：NK300 configuration error.
    An error occurred when you attempted to XXXX. 中文：关闭本地文件VERSION.INI文件出错。
    英文：An error occurred when you attempted to close the local Version.ini file.    		 当“错误”前面的定语太长时，使用此句型。
    不允许进行XXXX操作 XXXX is not allowed.
    XXXX cannot be-ed.    		 中文：该项数据不允许输入引号。
    英文：The quotation marks are not allowed in the data.    		 1.建议采用被动语态，尽量少使用第二人称为主语的主动语态。（如： You are not allowed to do XXXX. 或 You cannot do XXXX.）
    2.注意cannot不要写成can’t或can not.
    XXXX已经达到最大值 XXXX reached the maximum.
    或XXXX have reached the maximum.    		 中文：提交的短消息数已经到达最大值。
    英文：The number of submitted SMs has reached the maximum.    		 在同一产品中，其提示语的时态应该统一。
    XXXX已经被取消、删除、改变 XXXX is cancelled (deleted, changed).
    或XXXX have been cancelled (deleted, changed).
    比较abort的用法，如command（operation，processing）aborted。    		 中文：登录被取消！
    英文：The re-login is cancelled.
    中文：目标节点1%的状态已经改变。
    英文：The state of the destination node 1% has been changed.    		 在同一产品中，其提示语的时态应该统一。
    在用户手册中，尽量不要用abort，可根据意思选用exit，halt，interrupt，quit，stop等词。

    标签

    针对不同类型的返回信息，使用不同的标签进行标识。

    警告

    适用于描述如果错误操作，则可能导致操作人员伤亡或设备损坏。

    Warning

    尚未建立机械坐标系的情况下，不能正确进行行程极限检查。请在通电后重新建立连接后，执行了回参考点的状态下执行高速程序检测。

    注意

    适用于描述如果错误操作，则有可能导致操作无法完成。

    Caution

    在向外部I/O装置存取期间，请勿切断CNC的电源或拆除外部I/O装置。否则将会损坏外部I/O装置。

    备注

    适用于除警告和注意以外的补充说明。

    Note

    USB存储器的可用空间不多或者文件数量较多时，向USB存储器的文件输入/输出操作有时需要一定的时间。这种情况下，请删除USB存储器中不需要的文件后再度进行操作。

    在编写引用及告警标签时， 须保证与后续段落间保持一个空行。
