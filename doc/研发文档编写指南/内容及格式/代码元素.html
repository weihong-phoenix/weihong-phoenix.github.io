    代码元素

    代码元素 指的是编程语言关键字、类名、方法、属性名称、变量等，寻址 则是我们产品中比较典型的代码元素。

    说明： "什么内容可以称为代码，并不总是显而易见。 例如，NuGet 包名称应被视为代码"，引用自微软的文档指南。

    单个代码元素

    在文本中引用上下文代码块中的命名参数、变量、属性、方法、类和语言关键字时，一般是一行中的单个元素（单词），使用内联 `` 格式，且前后一个空格。

    正例 反例
    手轮设备可以通过 #HandwheelDevice.Direction 来控制手轮是正转还是反转。 手轮设备可以通过 #HandwheelDevice.Direction 来控制手轮是正转还是反转。
    Note

    对寻址本身的定义时则不需要。比如 Phoenix 子系统的 SRS 中定义信号、参数时，其寻址不需要使用 `` 。

    正例:

    手轮设备信号，脉冲计数

    项 指
    寻址 #HandwheelDevice.PulseCount
    标题 脉冲计数
    描述 手轮设备摇动所产生的脉冲计数。
    类型 int

    反例：

    手轮设备信号，脉冲计数

    项 指
    寻址 #HandwheelDevice.PulseCount
    标题 脉冲计数
    描述 手轮设备摇动所产生的脉冲计数。
    类型 int

    说明： 此处非正文中引用寻址，因此不需要另外加 `` 进行标识。

    文档中的代码块

    文档中需要引用大段代码时，使用代码块，示例如下：

    var host = new NcStudioHost();
host.CreateStandardBuilder(args);
host.StandardBuild();
