    要描述功能的具体说明。这部分我们参考的是 FANUC 的连接说明书，在编写文档的时候，更多从功能的角度出来来思考文档的提供方式，这里只是初步提供了一个建议的例子。

    【必须】功能名称

    功能的名称，不需要上下文就能准确识别功能含义。

    【必须】概述

    主要介绍该功能是什么，有什么用途、目的，介绍该功能执行的基本动作过程。一般先整体描述功能，然后再详细叙述。如果有需要的话，可以按照子功能的方式列举。

    除此之外可以选择性地介绍功能的背景、或者该功能相关的其他信息帮助读者理解。

    这部分会对于其中的功能做详细的描述，以表格的方式记录。 为了一致性，统一使用一样的表格，参数中不涉及的项，它的值写“-”。

    项 值
    寻址 访问方式，一般是以寻址的方式呈现
    名称 信号名称
    描述 信号的功能描述，目的价值，使用方法。
    读写 只读，只写，读写
    类型 信号的类型：bool, int, double, string
    范围 可以使用[],()来表示数字的范围
    对于bool类型，需要说明
    对于枚举类型，如果是 10 个以内，可以在表格中写下来，用<br>换行，超过 10 个的另外补充一个表格
    单位 数字的单位
    精度 参数的精度
    默认值 参数的默认值
    权限 用于指定参数权限，当前只有操作员和制造商
    分类 用于指定参数是轴参数、加工参数、规划参数、补偿参数、刀具参数、其他参数。。。
    参数类别 用于指定参数类别是 机床参数 还是 控制参数
    备注 一些补充的说明；比如说参数和其他有关联性；比如说一些特殊的用法等。

    示例功能：旋转轴指定

    （部分内容是杜撰的，作为格式的示例）

    概述

    可以通过参数，设定每个轴是直线轴还是旋转轴。旋转轴包括两种类型，可以通过参数设定为直线型旋转轴和旋转型旋转轴。

    问题点：需求、场景、价值应该在这里体现吗？如果有的话，可以记录。因为很多时候我们只是描述功能，所以并不需要。

    通过 #Axes{AN}.RotaryAxis 参数设定，可以开启循环显示功能。使用循环功能时，可以设定每转动一圈的移动量，坐标值的显示值会显示在【0，360】范围内，可以防止旋转轴的坐标溢出。

    绝对指令的情况下，通过参数 最短路径（#Pmt.Ch{CN}.Axis.RevShortPath[{LN}]），可以设定按照最短的移动方式移动到终点。

    绝对指令的情况下，通过参数 符号决定方向（#Pmt.Ch{CN}.Axis.SignDeterminesDir[{LN}]），可以设定移动的方向与指令的正负号相同。

    最短路径和符号决定方向不可以同时开启，如果开启了，按照最短路径处理。

    如果使用增量指令，最短路径和符号决定方向都不会生效，按照实际的移动值进行移动。

    Note

    该功能只有在 #Axes[{AN}].RotaryAxis 开启后有效

    接口

    是否循环显示中

    项 值
    寻址 #Axes[{AN}].IsRotaryAxisDisp
    名称 是否循环显示中
    描述 开启循环显示功能后，工件坐标值的显示范围在[0, 360) 之间。
    读写 只读
    类型 bool
    范围 true: 循环显示功能开启中；
    false: 循环显示功能关闭中
    默认值 false
    备注 -

    参数

    旋转轴循环显示

    项 值
    寻址 #Axes[{AN}].RotaryAxis
    名称 旋转轴循环显示
    描述 开启循环显示功能后，工件坐标值的显示范围在[0, 360) 之间。
    读写 读写
    类型 bool
    范围 true: 循环显示功能开启中；
    false: 循环显示功能关闭中
    默认值 false
    备注

    最短路径

    项 值
    寻址 #Pmt.Ch{CN}.Axis.RevShortPath[{LN}]
    名称 最短路径
    描述 按照最快移动到终点的方式进行移动，每次运动的距离不会超过 360°。
    读写 读写
    类型 bool
    范围 true: 启用；
    false: 不启用
    默认值 false
    备注 该功能只有在#Axes[{AN}].RotaryAxis启用后有效。

    符号决定方向

    项 值
    寻址 #Pmt.Ch{CN}.Axis.SignDeterminesDir[{AN}]
    名称 符号决定方向
    描述 按照最快移动到终点的方式进行移动，移动的方向和符号相同。正的时候按照顺时针移动，负的时候按照逆时针运动。每次运动的距离不会超过 360°。
    读写 读写
    类型 bool
    范围 true: 启用；
    false: 不启用
    默认值 false
    备注 该功能只有在#Axes[{AN}].RotaryAxis启用后有效，并且如果#Pmt.Ch{CN}.Axis.RevShortPath[{AN}]启用，该功能无效。

    旋转轴一圈移动量

    项 值
    寻址 #Axes[{AN}].CycleDistance
    名称 旋转轴一圈移动量
    描述 旋转轴每旋转一圈，移动的角度量。
    读写 读写
    类型 int
    范围 [0, 999999]
    默认值 360
    备注 该功能只有在#Axes[{AN}].RotaryAxis启用后有效。

    报警

    编号 类型 内容 说明
    10000 警告 加工过程中不可以设定最短路径 原因分析：在加工过程中修改了最短路径。
    解决办法：请在非加工状态下修改该参数。

    诊断

    项 值
    寻址 #Axes[{AN}].RotaryHistory
    名称 是否循环显示中诊断信息
    描述 记录了从开机到当前时间的旋转轴参数的修改次数。
    读写 只读
    类型 int
    范围 [0, 99999]
    默认值 0
    备注 关机后自动清零。超过最大值后变为 0 。

    示例

    旋转轴范围显示在 0° ~ 720°

    由于客户使用了放大机，所以旋转轴旋转 2 圈才会真正移动一圈，所以设定轴的显示范围在 0° ~ 720°。

    1. 将 #Axes[{AN}].RotaryAxis设置为 true
    2. 将 #Axes[{AN}].CycleDistance设置为 720
    3. 将 #Pmt.Ch{CN}.Axis.RevShortPath[{AN}] 设置为 true
    4. 执行刀路 X1080，最终顺时针旋转 1.5 圈，绝对坐标值为360°
